# Lab1: Creación de un Blog

Este proyecto es un blog sencillo realizado como actividad de laboratorio 1. Como ejemplo, hemos explicado el problema ferroviario español, utilizando tecnologías web básicas: **HTML**, **CSS** y **JavaScript**.

## Estructura del proyecto

```
lab1/
│
├── index.html        # Página principal del blog
├── styles.css        # Estilos personalizados para el blog
├── script.js         # Lógica para la navegación entre entradas
└── images/           # Carpeta con las imágenes usadas en las entradas
```

## Tecnologías utilizadas

- **HTML5**: Estructura semántica de la página y las entradas.
- **CSS3**: Estilos modernos, responsive y visualmente atractivos.
- **JavaScript**: Navegación dinámica entre las entradas del blog.

## Características

- **Navegación dinámica**: Solo se muestra una entrada a la vez, seleccionable desde el menú superior.
- **Entradas separadas**: Cada entrada es una sección independiente, con su propio contenido e imagen.
- **Imágenes ilustrativas**: Cada entrada incluye una imagen relevante, ubicada dentro de la propia sección.

## Cómo visualizar el blog on premise

1. Descarga o clona el repositorio en tu equipo.
2. Abre la carpeta del proyecto.
3. Haz doble clic en `index.html` para abrirlo en tu navegador.
   - **Opcional:** Si usas Visual Studio Code, puedes instalar la extensión "Live Server" y abrir el archivo con ella para ver los cambios en tiempo real.
